# 🔬 ResearchMind – Multi-Agent AI Research System

ResearchMind is a **multi-agent AI system** that automates the process of researching a topic, extracting insights, generating a report, and critically evaluating it.

It combines multiple specialized AI agents into a **structured pipeline** to deliver high-quality research outputs.

---

## 🚀 Features

- 🔍 **Search Agent** – Finds recent and relevant information  
- 📄 **Reader Agent** – Extracts deep insights from sources  
- ✍️ **Writer Chain** – Generates a structured research report  
- 🧐 **Critic Chain** – Reviews and improves the report  
- 🎯 **End-to-End Pipeline** – Fully automated research workflow  
- 💻 **Interactive UI** – Built with Streamlit
